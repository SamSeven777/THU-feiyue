# 个人网站/作品集

个人网站是一个展示自己的好机会，适合展示图片、视频等多媒体材料，可以更加立体地展现自己。

搭建个人网站可以使用建站工具，可以套用模板，也可以自己手搓。

个人网站是可以随时更新的，在其他文书材料中只需要附上网址，提交申请后还可以继续完善与更新内容。

建站工具：

- [Wix](https://zh.wix.com/)

[GitHub网站模板](https://github.com/topics/website-template)
